# 주간업데이트 워크플로우

매주 월요일 분석이 끝난 후 실행하는 콘텐츠 보드 반영 자동화 프로세스입니다.

---

## 🏃 실행 단계

1. **분석 결과 종합:** `주간패턴` 리포트와 `미달분석` 리포트를 취합합니다.
2. **콘텐츠 보드 수정 타겟팅:** 다음 주(혹은 남아있는 예약 큐)의 발행 예정 글들 중 수정이 필요한 대상을 식별합니다.
3. **콘텐츠 최적화 반영:**
   - 잘 된 패턴의 첫 줄 구조(예: 숫자로 시작하기)를 수정 대상 글에 대입하여 재작성.
   - 실패 원인으로 판명된 주제나 키워드는 보드에서 제외하거나 다른 셀럽 키워드로 긴급 교체.
   - 발행 시간대 스케줄 재배치.
4. **최종 업데이트 사항 요약:** 변경된 사항을 기록하여 사용자에게 전달합니다.
