## 🏃 실행 단계

### Step 1. 오늘 날짜 및 콘텐츠 확인
- 현재 시스템 날짜(예: 7월 3일)에 해당하는 행을 `콘텐츠보드.md`에서 조회합니다.
- 예: 7월 3일 -> 다이어트 (여름 샐러드 베스트 3) / 풀링 유형 / `@레퍼런스.md #3` 참조.

### Step 2. 글쓰기 규칙 및 패턴에 맞춰 초안 재구성
- **첫 줄:** 질문이나 숫자, 반전으로 시작하여 어그로 확보.
- **본문:** 짧고 간결하게 작성하며, 마지막 문단에서 핵심 핵심을 가림.
- **댓글:** 본문에서 가린 핵심 내용을 상세히 기술.

### Step 3. API 전송
- 서버 API `POST /api/post`를 사용해 본문을 발행합니다.
- 발행 완료 후, 첫 번째 댓글(Reply) API를 호출하거나 대시보드를 통해 수동 또는 API 지원 범위에 맞춰 댓글을 순차적으로 등록합니다.
- 대시보드의 '발행 이력'에 결과가 실시간 기록됩니다.
