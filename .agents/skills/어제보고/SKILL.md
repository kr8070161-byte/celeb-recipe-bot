# 어제보고 워크플로우

이 워크플로우는 전날 발행된 쓰레드 글의 도달 및 반응 데이터를 분석하여 성과 지표를 보고합니다.

---

## 📊 성과 분류 기준

1. 🔥 **잘 된 글 (Success):** 조회수 **5,000회 이상**
2. 🏆 **진짜 잘 된 글 (Super Success):** 공유 및 리포스트 **10회 이상**
3. ⚠️ **분석 필요 (Underperforming):** 조회수 1,000회 미만

---

## 🏃 실행 단계

1. **데이터 수집:** Threads API 또는 대시보드 백엔드 분석 로그를 통해 전날(어제) 올라간 게시물의 조회수(Views), 공유/리포스트 수(Shares)를 확인합니다.
2. **성과 필터링:**
   - 조회수 >= 5000 인 글 리스트 추출
   - 공유수 >= 10 인 글 리스트 추출
3. **결과 요약 보고:**
   - 발행한 글 본문 요약 및 링크
   - 조회수, 공유수 수치 표기
   - 잘 된 글 / 진짜 잘 된 글 분류 결과 한 줄 리포트 작성
